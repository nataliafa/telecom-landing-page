# Telecom Landing Page

Features:

- Routing
- Light/dark theme with saving to local storage
- Slider without the use of third-party libraries
- Rate cards with switches

## Interface

![light_theme](images/telecom.gif)

## Build & Run

Install dependencies:

```shell
npm install
```

Run for development:

```shell
npm run start
```

Production build:

```shell
npm run build
```
